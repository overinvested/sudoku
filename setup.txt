To start the flask server (Windows cmd):

> set FLASK_APP=puzzle
> flask run

If you are on mac/linux or using powershell, please see flask documentation, replace 'hello' with 'puzzle': https://flask.palletsprojects.com/en/2.1.x/quickstart/

Go to: localhost:5000/puzzle/<input puzzle here> in your browser

Please input the puzzle in the same format as the pdf writeup
Puzzles are assumed to be 9x9

i.e.,

((nil nil nil nil nil 6 nil 8 nil)
(3 nil nil nil nil 2 7 nil nil)
(7 nil 5 1 nil nil 6 nil nil)
(nil nil 9 4 nil nil nil nil nil)
(nil 8 nil nil 9 nil nil 2 nil)
(nil nil nil nil nil 8 3 nil nil)
(nil nil 4 nil nil 7 8 nil 5)
(nil nil 2 8 nil nil nil nil 6)
(nil 5 nil 9 nil nil nil nil nil))

All of the parentheses and spaces will be filtered out, nils will be interpreted as None
